# このソフトウェアについて #

WndProcClass201607260715は私個人が学習目的で作成したソフトウェアです。

WindowプログラミングにおけるWndProc関数をC++のclassでラップして実装できるようにしました。
また、Visual C++ 2010 Expressにて複数プロジェクトの作り方を確認しました。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express
